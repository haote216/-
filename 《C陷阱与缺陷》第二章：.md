1.任何C变量的声明都由俩部分组成：类型以及一组类似表达式的声明符。

  float f, g; 这个声明的含义：当对其求值时，表达式f和g的类型为浮点数类型。

  float ((f))；这个声明的含义：当对其求值时，((f))的类型为浮点数类型，由此可以推知f也是浮点数类型。

  float ff(); 这个声明的含义：当对其求值时，表达式ff()的结果是一个浮点数，也就是说，ff是一个返回值为浮点类型的函数。

  float *pf;这个声明的含义： *pf是一个浮点数，也就是说pf是一个指向浮点数的指针。

  float *g(),(*h)(); 表示 *g()与(*h)()是浮点表达式，g是一个函数，该函数的返回值类型为指向浮点数的指针。同理，h是一个函数指针，h所指向函数的返回值为浮点类型。   //  （）结合优先级高于*。

  float (*h)();表示h是一个指向返回值为浮点数类型的函数的指针。
  因此，(float (*)())表示一个“指向返回值为浮点数类型的函数的指针”的类型转换符。

练习  2-1.C语言允许初始化列表中出现多余的逗号，例如：
int days[] ={31,28,31,30,31,30,31,31,30,31,30,31,};为什么这种特性是有用的？

```
我们可以把上例的缩排格式稍作改动如下：
int days={
    31,28,31,30,31,30,
    31,31,30,31,30,31,
}
现在我们可以很容易看出，初始化列表的每一行都是以逗号结尾的。正因为每一行在语法上的这种相似性，自动化的程序设计工具（例如，代码编辑器）才能更方便的处理很大的初始化列表。
```
