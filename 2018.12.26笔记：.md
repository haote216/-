```
1.字符串时以'\0'作为结束标志。
strlen函数返回的是在字符串中'\0'前面出现的字符个数（不包含'\0'）。参数指向的字符串必须要以'\0'结束。stelen函数的返回值是size_t，是无符号的。
```

```
2.char str1[50]="hello world";
  char str2[50]={0};
  char *str3="hello hello hello";
  strcpy(str2,str1);
  strcpy(str3,str1);
  第二次拷贝出错，这是一个指向常量字符串的指针，类似于常量指针，其指向的内容不能被修改。
  const char *p=&a   或 char const *p = &a
  常量指针是因为用const保护了，常量字符串时字符串在静态内存区。
  
  定义字符串常量：char *a ="i love you";
  定义字符串变量：char a[]="i love you";
```

```
3.常量指针和指针常量的区别：
（1）常量指针：指向常量的指针，它不能指向变量，指向的内容不能修改。但是指针本身不是常量，从而可以指向另一个常量。
  int b,c;   int const * a;
  a=&b; a=&c;(正确)
  *a=20;（错误）
（2）指针常量：指针本身就是常量。它指向的地址是不可改变的，但是地址里的内容可以通过指针来改变。它指向的地址伴其一生，也就是说，指针常量在定义时必须同时赋初始值。
  int * const p =&a;
```
